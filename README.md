# rampyai web mode

This folder is a browser-based version designed to work on older iPhone Safari versions as far back as practical.

## Use
1. Put `logo.png` in this folder.
2. Run `node server.js` from this folder.
3. Open `index.html` on iOS 4/5/6 Safari.

## Notes
- Uses simple HTML/CSS/JS only.
- Settings are stored in `localStorage` when supported.
- The Node proxy hides your API key from the browser request.
